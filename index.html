My latest project

CineReview is a user-friendly anime and manga review website, crafted with Bootstrap for a sleek front-end, PHP for dynamic server-side functionality, and MySQL for secure database management. Users can easily register, providing a personalized experience with encrypted password storage for enhanced security. The platform allows browsing and searching for reviews on a variety of anime and manga titles, each featuring ratings, comments, and detailed critiques. Registered users can actively contribute by adding new titles to the database, promoting community engagement. The website boasts a social aspect, enabling users to comment on and like/dislike reviews. Rigorous testing with the JOSF program ensures reliability, and robust security measures, including password encryption and vulnerability prevention, enhance user data protection. CineReview is your one-stop destination for exploring, reviewing, and discussing anime and manga.
